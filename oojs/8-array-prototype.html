        8. [OOJS] Array Prototype dan High Order Function

        Photo by Chris Ried on Unsplash

        Setelah kita belajar sedikit tentang tipe data array, selanjutnya kita menyelam lebih jauh mengenai Array. Karena sebenarnya array sendiri cukup powerful dan juga luas dalam penggunaannya. Karena sejatinya elemen-elemen dasar dalam Javascript adalah object, maka Array juga termasuk object yang memiliki method dan juga property.

        # Property array

        3 Properti dasar array di javascript

        # 1. length

        Digunakan untuk menghitung panjang dari array.

        let buah = ["jambu", "strawbery", "apel", "mangga"];

// panjang array menggunakan property length
console.log(`Panjang array adalah: ${buah.length}`);

        # Prototype/Method umum array

        Method-method umum yang sudah ada secara default dalam javascript.

        # 1. push() dan pop()

        Method push digunakan untuk menambah elemen array di paling belakang elemen lainnya. Sementara pop sebaliknya, digunakan untuk menghapus elemen array paling belakang.

        let array1 = ["a", "b", "c"];
let array2 = ["x", "y", "z"];
console.log(array1);
console.log(array2);

// Menambah elemen array1 dengan 'd' di belakang
array1.push("d");
// Menghapus elemen array2 di belakang
array2.pop();
console.log("Setelah dilakukan operasi");

console.log(array1);
console.log(array2);

        # 3. splice()

        splice digunakan untuk mengubah isi array dari index tertentu misalkan ditengah-tengah array, kemudian bisa juga menghapus dan menambahkan elemen di index tersebut.

        Sintaks: array.splice(index, deleteCount, item)
        index : berisikan mulai dari indeks keberapa array diubah
        deleteCount : berapa jumlah elemen yang akan dihapus mulai dari index tersebut, 0 jika tidak ingin menghapus
        item : elemen yang ingin ditambahkan

        let tempatDuduk = [`imam`, `fadhli`, `willy`, `diky`];
console.log(`Mahasiswa TI tengah duduk dengan tenang`);
console.log(tempatDuduk);

// Menambahkan munib ditengah2 fadhli dan willy
tempatDuduk.splice(2, 0, `munib`);
console.log(`Tiba-tiba munib datang menyerobot`);
console.log(tempatDuduk);
// index setelah fadhli adalah 2, dan 0 elemen yang akan dihapus
// lalu tambahkan munib

// Menghapus fadli
tempatDuduk.splice(1, 1);
console.log(`Fadhli pun pergi`);
console.log(tempatDuduk);
// splice diatas hanya 2 argumen karena argumen ketiga bersifat opsional
// index fadhli adalah 1 dan 1 elemen yang akan dihapus

        # 4. join()

        join merupakan lawan dari method split milik tipe data string. Dimana jika split memecah string menjadi array, sementara join menggabungkan array menjadi string. Argumen yang ditangkap adalah separator yaitu simbol yang akan digunakan sebagai pemisah atau penyatu.

        let kode = ["abo", "go", "boga"];
let koordinat = [12, 23, 35, 756];
let stats = [`agi`, `str`, `def`, `vit`];

console.log(kode);
console.log(koordinat);
console.log(stats);

console.log(`Setelah join`);

//tanpa argumen pemisah/penyatu
kode = kode.join();

//dengan argumen titik
koordinat = koordinat.join(".");

//dengan argumen spasi | spasi
stats = stats.join(` | `);

console.log(kode);
console.log(koordinat);
console.log(stats);

        # 5. includes()

        # 6. find()

        find hampir mirip dengan method includes, perbedaannya adalah jika includes hanya mengecek nilai sedangkan find mampu melakukan evaluasi nilai untuk melakukan pengecekan lebih dalam. find juga mengembalikan nilai true jika ada nilai yang sesuai dengan evaluasi dan false jika tidak ada.

        //array yang berisikan object
let mhs = [
	{ nim: 123, nama: "Yama Morikawa" },
	{ nim: 321, nama: "Tadano Bukatsu" },
	{ nim: 456, nama: "Kurasa Takada" },
	{ nim: 542, nama: "Tasegawa" },
];

//menggunakan find untuk mencari mhs dengan nim 456

let takada = mhs.find((mahasiswa) => mahasiswa.nim == 456);
console.log(takada);

        # 7. every()

        Method every hampir sama dengan find hanya saja akan mengembalikan nilai true jika semua elemennya memenuhi evaluasi, sedangkan pada method find mengembalikan nilai true jika salah satunya saja.

        let angka = [1000, 2000, 3000, 4000];

function apakahBesarSemua(angka, indeks, array) {
	return angka >= 500;
}

let cekAngka = angka.every(apakahBesarSemua);

console.log(`Apakah array angka semuanya lebih besar dari 500? ${cekAngka}`);

        # High Order Function

        High Order Function adalah function yang bisa di chain atau digabungkan setelahnya menggunakan simbol . (titik) tanpa memperhatikan urutan dan menggunakan callback function sebagai parameternya.

	console.log(`Elemen indeks ke ${indeks}, nilainya adalah ${elemen}`)

        # Membuat prototype sendiri

        Misalkan method array untuk memasukakan nilai ditengah array yaitu splice terlalu sulit dihafal karena 3 parameter dan keywordnya bisa tertukar dengan method slice. Maka kita bisa buat prototype kita sendiri dengan menggunakan property prototype milik array diatas.

        // Array adalah object global dari array
// Tidak bisa menggunakan arrow function
// Karena terdapat keyword this di dalamnya

Array.prototype.masukTengah = function(indeks, isi) {
	return this.splice(indeks, 0, isi);
};

// Diatas adalah cara pembuatan prototype array,
// yg bisa dipakai di array mana saja
let array1 = ["b", "c", "d"];
let array2 = ["w", "x", "y", "z"];

console.log(array1);
console.log(array2);

console.log(`Setelah dimasukkan: `);
array1.masukTengah(1, "Halo");
array2.masukTengah(3, "Yahooo");

console.log(array1);
console.log(array2);

        # Penggunaan High Order Function

        Salah satu kegunaan high order function yaitu bisa melakukan chaining atau merantai langsung eksekusi function tanpa perlu disimpan ke variable terlebih dahulu.

        //Dibawah ini adalah contoh chaining
//untuk menghilangkan angka dari sebuah string
//dan kapitalisasi tiap huruf

let kalimat = `aku 3 adalah seorang kapiten 10`;
let terFilter = kalimat
	.split(" ")
	.filter((el) => isNaN(el / 2))
	.map((el) => el[0].toUpperCase() + el.substring(1))
	.join(" ");

console.log(terFilter);
