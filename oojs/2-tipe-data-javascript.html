        # Apa itu tipe data?

        Tipe data adalah jenis-jenis dari data. Misalkan pakaian, memiliki banyak tipe-tipe data pakaian seperti baju, celana, topi dan sebagainya. Tipe data dalam javascript adalah jenis-jenis data yang bisa disimpan ke dalam sebuah variabel.

        Isi dari sebuah variable adalah sebuah data, nah data itu beragam tipenya.

        # Tipe data

        # 1. Number

        Yang pertama adalah number, tipe data number merupakan data yang berbentuk angka baik itu bilangan bulat (integer) maupun bilangan pecahan (float). Cara mengisikan number ke dalam sebuah variable adalah dengan memasukkan langsung nilai dari angka tersebut.

        let btc = 100;
// Deklarasi variable btc dengan nilai 100
// berupa tipe data number
        1
        2
        3

        Untuk membuktikan apakah benar variable btc di atas merupakan sebuah number kita bisa menggunakan sebuah fungsi typeof(namaVariablenya).

        let btc = 100;

console.log(typeof btc);
// Tampilkan ke layar tipe data btc ke layar
// Maka akan menghasilkan output: number
        1
        2
        3
        4
        5

        # 2. String

        String merupakan tipe data karakter, baik itu huruf, simbol, kalimat bahkan juga angka. Pemberian nilai string bisa dilakukan dengan mengapit nilai menggunakan tanda:

        1. Petik dua ("")
        2. Petik satu ('')
        3. Backtick (``)
        let namaDepan = "The Hash";
let namaTengah = "Slinging";
let namaBelakang = `Slasher`;
// Perhatikan tanda kutipnya

console.log(typeof namaDepan);
console.log(typeof namaTengah);
console.log(typeof namaBelakang);

        # 3. Boolean

        Tipe data yang digunakan untuk menandai kondisi dalam program apakah bernilai true atau false. Tipe data ini akan sangat berguna dalam perbandingan atau pengecekan suatu nilai.

        let capekNulis = true;

if (capekNulis) console.log("Istirahat");
if (!capekNulis) console.log("Lanjut nulis");

        # 4. Array

        Tipe data array digunakan untuk menyimpan data lebih dari satu seperti layaknya sebiuah rak sepatu. Array bisa menampung string, number, boolean, object dan juga array laig (array dalam array). Array adalah salah satu tipe data favorit saya karena fleksibilitas dan kemudahan penggunaan untuk menyimpan banyak nilai. Penulisan array dibuka dengan kurung [, dipisahkan dengan , dan ditutup dengan ].

        let buah = ["mangga", "apel", "duren"];
let campur = [012, "mangga", true, { nama: "mimi" }];

console.log(buah[0]);

        # 5. Object

        Object merupakan tipe data yang dibuat untuk menyimpan sebuah objek. Semua yang ada dalam javascript dibangun dengan menggunakan tipe data object. Ada berbagai macam cara pendeklarasian object, namun untuk saat ini yang perlu kita ketahui adalah deklarasi object secara langsung.

        Penulisan object dibuka dengan kurung { dan ditutup dengan }, kemudian berisi pasangan antara key dan value. Object juga bisa menampung string, number, boolean, array dan juga object (Object dalam object) serta fungsi (yang akan disebut dengan method)

        let Manusia = {
    nama: 'murabito b' //string
    mata: 2, // number
    bisaMelihat: true, // boolean
    elemenMata: ['pupil', 'kornea', 'retina'], // array
    melihat(){ //method
        console.log('Halo, namaku ' + this.nama)
        console.log('aku bisa melihat')
    }
}

console.log(Manusia.mata)
console.log(Manusia.melihat())

        # 6. Function

        Function merupakan fungsi. Digunakan untuk mendefinisikan sebuah aksi dalam program. Tujuan dibuatnya sebuah function adalah supaya kita bisa memanggil sebuah fungsi tanpa mendefiniskan ulang, menandai bagian dalam program dan mempermudah pemrosesan program.

        Penulisan function terdapat dua macam yaitu function declaration dan function expression. Untuk sekarang kita hanya akan mempelajari tentang function declaration. Penulisan function declaration adalah dengan menuliskan keyword function namaFunctionnya(){ //kode disini }. Cara pemanggilan function adalah dengan menuliskan namaFunctionnya().

        Keyword return di dalam function digunakan untuk mengembalikan nilai dan mengakhiri eksekusi kode di dalam function itu sendiri. Jika kita tidak mendefiniskan return maka secara default nilai yang akan dikembalikan adalah undefined.

        let namaDepan = "Osamu ";
let namaBelakang = "D";

function tampilkanNama() {
	console.log(namaDepan + namaBelakang);
}

tampilkanNama();
// Memanggil function

        # 7. Lainnya

        Tipe data lainnya yang patut dipelajari adalah undefined, null, NaN dan infinity atau -infinity.

        1. Undefined
          Tipe data untuk suatu variable yang belum memiliki nilai
        2. Null
          Tipe data untuk sebuah Object yang kosong
        3. NaN (NotaNumber)
          Tipe data yang dihasilkan apabila kita melakukan operasi aritmatika antara string dengan number
        4. infinity & -infinity
          Tipe data untuk menyatakan nilai tak terhingga, semisal pembagian dengan nol.

        # Latihan

        Oke, di atas merupakan 7 tipe data dasar yang wajib dipelajari dan sebenarnya masih ada tipe data lanjutan yang lain seperti date, LinkedList, NodeList dan seterusnya. Untuk saat ini ketujuh tipe data diatas hampir sama di setiap bahasa pemrograman lainnya.

        Karena kita akan masuk ke dalam dunia OOP, maka dari itu kita harus terbiasa menggunakan tipe data Object.

        let Kubus = {
	sisi: 10,
	tampilkanVolume(sisiMasukan = this.sisi) {
		// sisiMasukan merupakan variable dinamis atau
		// biasa disebut parameter, yang nilainya sesuai
		// dengan inputan user ketika memasukan data di dalam kurung
		// sisiMasukan = this.sisi berarti parameter sisi masukan akan
		// diisi default dengan nilai sisi apabila user tidak mengirimkan
		// argumen ke dalam parameter
		this.sisi = sisiMasukan;

		// Definisikan rumus
		let rumus = this.sisi * this.sisi * this.sisi;

		// Tampilkan rumus
		console.log(
			"Volume kubus dengan nilai sisi " + this.sisi + " adalah ",
			rumus
		);
	},
};

console.log(Kubus.tampilkanVolume());
// Kode diatas akan menghasilkan output: 1000
// karena kita tidak mengirimkan argumen ke parameter
// sehingga nilai default sisi adalah 10

console.log(Kubus.tampilkanVolume(12));
// Kode diatas mengirimkan data 12 dimana di dalam
// method tampilkanVolume, data tersebut akan ditampung
// di parameter sisiMasukan
