      # Function

      Setelah mempelajari sedikit mengenai function dalam materi tipe data, sekarang kita akan menyelam lebih jauh mengenai function. Function atau fungsi digunakan untuk mendefinisikan suatu aksi atau blok program dimana aksi tersebut bisa kita panggil berulang-ulang (reusability).

      Function di dalam sebuah Object dinamakan Method

      Struktur dasar sebuah deklarasi function adalah sebagai berikut:

      # Function declaration

      // Diawali dengan keyword 'function'
// kemudian namafungsi diikuti kurung ()
// lalu kurung kurawal {}
// diakhiri dengan return

function namaFungsi() {
	// Kode program disini
	return;
}

      Terdapat dua cara pendeklarasian function, yaitu Function Declaration dan Function Expression. Function diatas merupakan pendeklarasian function declaration. Function expression merupakan function yang disimpan ke dalam sebuah variable sebagai sebuah nilai referensi. Biasanya disimpan ke variable const.

      # Function expression

      Function expression menyimpan hasil return dari eksekusi ketika dipanggil ke dalam variable pembentuknya.

      // function diisikan kedalam variable nama fungsi
const namaFungsi = function() {
	// Kode program disini
	return;
};

      # Return

      Keyword return digunakan untuk mengembalikan nilai serta mengakhiri eksekusi dari function tersebut. Nilai yang dimaksud adalah nilai yang telah diproses oleh fungsi tersebut.

      function halo() {
	return "selamat datang";
}

// Fungsi diatas apabila dipanggil akan
// mengembalikan nilai "Selamat datang
halo();

      Tingkah laku return

      1. Bersifat opsional
        Jika tidak dispesifikasikan akan mengembalikan nilai default yaitu "undefined"
      2. Hanya bisa mengembalikan satu nilai
        Nilai lebih dari satu dikembalikan lewat array atau object
      3. Bisa mengembalikan fungsi dan operasi
      4. Return dalam satu fungsi bisa lebih dari satu
      const kurangi = function(a, b) {
	if (b > a) return "pengurangan negatif";
	return a - b;
};
// Fungsi diatas merupakan function expression
// Memiliki dua buah return sesuai pengkondisian

// Kode dibawah akan menghasilkan nilai namun tidak
// ditampung ke variable
kurangi(5, 10);
kurangi(7, 3);

// Kode dibawah akan menyimpan hasil return
// kedalam variable angka1
let angka1 = kurangi(5, 10);
console.log(angka1);

      # Parameter dan argumen

      Parameter merupakan variable yang berada dalam tanda kurung () setelah nama function. Digunakan untuk menangkap argumen ketika function dipanggil. Parameter dan argumen digunakan untuk menghasilkan program atau suatu fungsi yang dinamis.

      // Function halo akan menerima 1 argumen
// yang akan diisikan ke parameter nama
function halo(nama) {
	console.log(`Halo, ${nama}`);
}

// Function waifu akan menerima 1 argumen
// yang akan diisikan ke parameter namaWaifu
const waifu = function(namaWaifu) {
	console.log(`Waifumu adalah: ${namaWaifu}`);
};

// Memanggil fungsi halo dan mengirimkan argumen "awal"
// Maka nama = "awal"
halo("awal");

// Memanggil fungsi waifu dan mengirimkan argumen "yukino"
// Maka parameter namaWaifu = "yukino"
waifu("yukino");

// Argumen "henky" dikirimkan ke fungsi selamatPagi

      Kita bisa menangkap argumennya menggunakan parameter default yaitu arguments. Jadi, tidak hanya untuk satu argumen, misalkan kita punya sebuah function yang hanya memiliki 1 parameter, kemudian kita mengirimkan 5 argumen sekaligus, maka 4 argumen sisa yang tidak ditangkap parameter akan disimpan kedalam parameter arguments.

      function tangkapBuah(buah1) {
	console.log(`Buah yang jatuh ditangkap: ${buah1}`);
	let buahJatuh = [...arguments].join(", ");
	// arguments merupakan bentuk array
	// Diatas digunakan Spread Syntax (...) untuk memecah
	// isi arguments ke dalam array baru lalu kita
	// gunakan method join(',') untuk
	// menggabungkan array dengan koma menjadi satu string

	console.log(`Seluruh buah yang jatuh: ${buahJatuh}`);
}

tangkapBuah("apel", "durian", "manggis", "karet");

      # Arrow function

      Arrow function merupakan cara penulisan function yang lebih singkat dari deklarasi function namun dengan karakteristik tertentu.

      1. Arrow Function tidak bisa diisi dengan keyword this
      2. Arrow function tidak membutuhkan keyword function

      Terdapat 3 penyederhanaan arrow function

      // Penulisan pertama
// Langsung menuliskan parameter (jika ada) dalam kurung lalu diikuti
// dengan tanda => dan kurung kurawal
// Apabila kode dalam blok program lebih dari satu baris
// harus menuliskan kurung kurawal dan return (opsional)
const salam = () => {
	let halo = `Halo!`; //baris pertama
	return halo; //baris kedua
};
console.log(salam());

// Penulisan kedua
// apabila satu parameter, kurung () tidak perlu
// apabila kode dalam { } hanya satu baris maka tanpa kurung { }
// Otomatis return tanpa keyword return
const salam2 = (nama) => `Halo, ${nama}`;
console.log(salam2("awal"));

// Penulisan ketiga
// apabila lebih dari satu parameter
// maka wajib dikurung
const salam3 = (nama, waktu) => `Selamat ${waktu}, ${nama}!`;
console.log(salam3("Rafiq", "Malam"));

      # Self anonymous function

      Self anonymous function merupakan function tanpa nama. Jika biasanya kita menyimpannya ke dalam variable const, self anonymous function tanpa adanya nama dan biasanya digunakan untuk callback function dan function yang langsung dijalankan. Self anonymous function erat juga kaitannya dengan SIFE/IIFE

      # IIFE/SIFE

      Immediately Invoke Function Expression / Self Invoking Function Expression merupakan fungsi yang memanggil dirinya sendiri langsung tanpa pemanggilan nama. Fungsi akan langsung dipanggil saat pembuatannya. SIFE/IIFE erat kaitannya dengan self anonymous function.

      // SIFE/IIFE dengan Self anonymous function
// Penulisan SIFE yaitu dengan membungkus function dengan ()
// lalu diikuti dengan () untuk wadah parameternya
// Jadi (function(){})(parameter)

((nama) => console.log(nama))("Willy");

      # Input Interrupt

      Merupakan prosedur untuk menghentikan sementara program, dengan tujuan menangkap masukkan user yang digunakan untuk menentukan kelanjutan laju dari program. Misalnya program menu pada restoran akan menuggu inputan dari user selesai sebelum memasak pesanan yang user pilih.

      Input interupt dalam node dapat diambil menggunakan module bernama readline-sync. Kali ini kita akan berkenalan sedikit dengan module atau package dalam node.js dan bagaimana cara menginstall suatu module.

      1. Buka CMD atau terminal kalian (pastikan terdapat internet)
      2. Ketikkan perintah npm install readline-sync
        arti dari perintah diatas adalah kita menggunakan npm (node package manager) untuk mengambil package bernama readline-sync
      3. Folder baru akan muncul dengan nama node_modules, disitulah tempat module-module per project yang kalian install akan disimpan.
      4. Pastikan folder node_modules berada dalam satu folder dengan file kerja kalian
      const bacaBaris = require("readline-sync");
// Perintah diatas untuk memasukkan module ke dalam file kita
// sehingga kita bisa mengakses properti dan method yang ada
// pada readline-sync

// Membaca inputan dan memasukkanya ke variable nama
let nama = bacaBaris.question("Siapa nama kamu? ");

console.log(`Nama kamu adalah ${nama}!`);
