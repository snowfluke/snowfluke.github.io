      0. [OOJS] Persiapan Lingkungan

      Photo by Safar Safarov on Unsplash

      # Persiapan

      Persiapan lingkungan pengembangan untuk mengikuti pembelajaran.

      # Text editor

      Text editor bisa berupa VSCode, Sublime, Notepad++ atau Atom.
      Saya merekomendasikan VSCode dengan keunggulan sebagai berikut:

      1. Built-in terminal
        Sudah ada terminal bawaan.

      2. Built-in emmet
        Sudah terinstall emmet/plugin auto-complete code. Jika menggunakan text editor lain seperti sublime, harus menginstall emmet secara manual.

      3. Banyak plugin
        Banyak plugin yang memudahkan proses pembuatan kode.

      # Node.js

      Download Node.js disini (opens new window) dengan versi LTS (Long Time Support) dengan versi 8 ke atas. Node.js digunakan sebagai runtime environment dan mengambil module-module Javascript yang diperlukan.

      Untuk instalasi, cukup run file .exe yang telah di download (saya asumsikan rekan-rekan semua memakai sistem operasi Windows). Klik Next sampai instalasi selesai.

      Kemudian apabila telah menginstall lakukan pengecekan dengan mengetikan perintah:

      node -v

      atau

      node --version

      Jika tidak muncul seperti contoh, berarti instalasi gagal, lakukan install ulang/restart jika diperlukan.

      # Browser

      Chrome/Firefox. Opsional, untuk keperluan manipulasi DOM jika diperlukan.

      # Devdocs.io

      Kunjungi situs DevDocs (opens new window) lalu download bagian Javascript, untuk mendownload dokumentasi semua fungsi, sintaks yang terdapat di Javascript, supaya bisa diakses secara offline.
