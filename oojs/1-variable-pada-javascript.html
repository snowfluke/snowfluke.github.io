            # Apa itu variable?

            Variable secara sederhana dapat dikatakan sebagai tempat penyimpanan nilai sementara. Variable merupakan konsep yang dipakai di seluruh bahasa pemrograman komputer. Dengan variable kita bisa membedakan antara data yang satu dan data yang lainnya.

            Analogi sederhana variable adalah nama kamu. Nama kamu bertujuan untuk menyimpan dirimu, pembeda dari yang lain dan identitas. Bayangkan apabila kamu tidak memiliki nama, bagaimana orang lain bisa mengenali, mengingat dan memanggilmu di kerumunan?

            Sama halnya juga di dalam pemrograman, variable diperuntukkan sebagai identitas dari data.

            Sebelum kita masuk ke dalam pendeklarasian variable, perlu diingat bahwa javascript menggunakan konvensi penamaan camelCase dimana huruf pertama huruf kecil dan huruf untuk kata selanjutnya adalah huruf besar. Serta huruf kapital di awal variable menanadakan variable tersebut adalah sebuah Object.

            // contoh camelCase naming convention
// variable umum
luasSegitiga; // benar
LuasSegitiga; // salah
LUASSEGITIGA; // salah
arrayBaru; // benar
hitungKedalaman; // benar

// variable untuk object
Segitiga; // benar
BangunRuang; //benar
bangunRuang; // salah
PintuRumah; // benar

            # Pendeklarasian variable dalam Javascript

            Dalam Javascript terdapat 3 cara untuk mendeklarasikan variable. Dimana masing-masing deklarasi variable punya tingkah laku/behavior yang berbeda-beda. Pendeklarasian tersebut yaitu var, let, dan const.

            Cara mengakses variable

            Cara mengakses variable adalah dengan menuliskan nama dari variable tersebut.

            # var

            Pendeklarasian variable dengan sintaks var membuat variable bersifat global, atau publik, bisa diakses dari mana saja dan bisa diganti nilainya atau di deklarasikan ulang.

            # Deklarasi ulang

            // Deklarasi var awal
var namaDepan = "maknum";

// Deklarasi ulang var
var namaDepan = "Imam";

            # let

            Pendeklarasian variable dengan sintaks let membuat variable bersifat block scope, tidak bisa diakses dari luar scope dan tidak bisa di deklarasikan ulang. Yang dimaksud dengan block adalah area dalam program yang diapit kurung{ dan }.

            # Deklarasi ulang

            // Deklarasi let awal
let nama = "willy setiawan";

// Deklarasi ulang let
// Kode dibawah akan menimbulkan error, karena let tidak bisa di deklarasikan ulang
let nama = "diky setiawan";

            # const

            Pendeklarasian variable dengan sintaks const kurang lebih sifatnya sama seperti let namun dikhususkan untuk variable yang nilainya tidak akan berubah-ubah atau tetap (konstanta).

            # Deklarasi ulang

            // Deklarasi const awal
const NAMA = "willy setiawan";

// Deklarasi ulang const
// Kode dibawah akan menimbulkan error,
// karena const tidak bisa di deklarasikan ulang

            # Tabel perbedaan var, let dan const

            Sintaks Scope Re-Deklarasi Fungsi Penulisan Re-Definisi
            var Global Bisa Variable publik var nama Bisa
            let Block Tidak bisa Variable private let nama Bisa
            const Block Tidak bisa Variable private const NAMA Hanya array & object

            Setelah mengetahui perbedaan ketiganya, lalu mana yang sebaiknya dipakai? dahulu javascript hanya menggunakan var di mana konsep block scope belum diimplementasikan seperti bahasa pemrograman lain. Hal ini menimbulkan kemanan Javascript yang rentan.

            Oleh sebab itu, apabila kita tidak benar-benar membutuhkan variable publik, gunakan selalu let untuk membuat variable. Kemudian untuk menyimpan nilai tetap, misalnya nilai PI, rumus, gunakan const, dan juga const dipakai untuk menyimpan function di materi-materi selanjutnya.

            # Contoh Program

            // Program menampilkan nama

// Deklarasi variable fadhli menggunakan let
let fadhli = "Nurfadhli Abdurrachman Hakim";
let henky = "Henky Fajar Syafani";

// sebuah function untuk menampilkan nama
// berdasarkan argumen yang dikirimkan
// function ini untuk sementara diabaikan terlebih dahulu
function tampilkanNama(nama) {
	console.log(nama);
}

// mengeksekusi function tampilkanNama dengan argumen
// berupa variabel fadhli dan henky
tampilkanNama(fadhli);
tampilkanNama(henky);

            # Latihan

            Apabila kamu telah menyiapkan lingkungan kerja seperti di materi sebelumnya (node.js, text editor dll). Selanjutnya adalah bagaimana kita menjalankan javascript.

            1. Buat sebuah folder baru, dengan nama misalkan latihanOOJS
            2. Buat file baru dengan ekstensi (.js), misalkan index.js
            3. Buka file tersebut di dalam code editor kalian

            Tips

            Jika kalian menggunakan VSCode, buka folder yang baru kalian buat menggunakan CMD/terminal, lalu ketikan code . untuk membuka folder di dalam text editor.

            1. Selanjutnya tuliskan kode Javascript dari contoh program ke file tersebut.
            2. Untuk menjalankan kode tersebut cukup ke CMD/terminal, masuk ke folder file tersebut lalu ketikkan node namafilenya.js

            Terminal VSCode

            Untuk VSCode lebih mudah, pada VSCode tekan Ctrl + ` (backtick) di sebelah angka 1. Lalu ketikkan node namafile.js
